## Огляд

Цього тижня Lightning Network досяг нової рекорду за потужністю, досягнувши приблизно 5,606 BTC. Основні криптовалютні біржі сприяли цьому зростанню, збільшивши свої оффчейн запаси біткоїнів, покращуючи загальну ліквідність мережі та змінюючи динаміку транзакцій біткоїнів для користувачів.

## Зростання потужності Lightning Network

Публічна потужність Lightning Network зазнала значного зростання, з повідомленнями про пік, що наближається до 5,637 BTC. Це збільшення відображає помітне покращення в порівнянні з попередніми рівнями та встановлює найвищу загальну потужність, зафіксовану на сьогодні. Основні біржі, такі як Binance та OKX, відіграли вирішальну роль у цьому розвитку, додавши біткоїни до каналів Lightning. Крім того, інші платформи, такі як Kraken та Bitfinex, розширюють свою підтримку мережі. Основна мета цих депозитів полягає в прискоренні процесів депозитів та виведень, одночасно зменшуючи комісії для клієнтів.

### Активність мережі та публічні вузли

Незважаючи на збільшення потужності, не спостерігається відповідного зростання кількості публічних вузлів або каналів у Lightning Network. Наразі кількість публічних вузлів становить приблизно 14,940, тоді як кількість публічних каналів становить близько 48,678. Це свідчить про те, що, хоча більше біткоїнів доступно в мережі, кількість суб'єктів, які забезпечують трафік, не зросла в тому ж темпі. Частина додаткової ліквідності зосереджена в більших, кастодіальних каналах, що експлуатуються біржами, які можуть управляти значними сумами без створення великої кількості нових публічних маршрутів. Це зосередження може ускладнити інтерпретацію метрик на блокчейні.

Кількість транзакцій та економія на комісіях на блокчейні ілюструють матеріальні переваги для користувачів, навіть у контексті стабільної графіки вузлів. Важливим показником для розгляду є частка трафіку біржі, що проходить через Lightning. Повідомлення вказують на те, що одна біржа успішно маршрутизувала близько 15% своїх біткоїн-транзакцій через Lightning, підкреслюючи значні операційні зміни на основних платформах.

## Нові варіанти використання та фінансування

Зростання потужності також викликало інтерес до фінансування та удосконалення протоколів. Tether очолив раунд фінансування, який залучив приблизно $8 мільйонів для стартапу, що фокусується на полегшенні платежів через Lightning Network, демонструючи зростаючий інтерес до транзакцій зі стейблкоінами в екосистемі. Крім того, анонс Taproot Assets v0.7 включає функції, такі як повторно використовувані адреси, повністю аудиторний обсяг активів та покращення для більших, більш надійних транзакцій. Ці оновлення закладають основу для потенційних трильйонів доларів у транзакціях на біткоїні та Lightning Network.

Оновлення протоколу впроваджуються для покращення здатності мережі підтримувати різноманітні типи платежів та транзакцій токенів. Ці вдосконалення свідчать про те, що Lightning Network еволюціонує за межі простих мікротранзакцій, з тестуванням застосувань для грошових переказів, платежів для торговців та трансакцій зі стейблкоінами.

## Від автора

Недавнє зростання потужності Lightning Network означає вирішальний момент для транзакцій біткоїнів, особливо коли основні біржі вносять більше ліквідності. Проте, різниця між збільшеною потужністю та стабільною кількістю вузлів викликає питання про децентралізацію мережі та наслідки залежності від кастодіальних каналів. Удосконалення можливостей протоколу, разом із зростаючим інтересом до різноманітних варіантів використання, свідчать про те, що Lightning Network перебуває на трансформаційному шляху.

## Вплив на крипто ринок

- Збільшена потужність Lightning Network може призвести до більш ефективних транзакцій біткоїнів.
- Участь основних бірж може покращити досвід користувачів через зменшення комісій та швидші часи транзакцій.
- Зростаюча частка трафіку біржі, що маршрутизується через Lightning, вказує на зміну в операційних практиках серед бірж.
- Нове фінансування для стартапів, орієнтованих на платежі через Lightning, свідчить про зростаючий інституційний інтерес до цієї технології.
- Оновлення протоколу можуть дозволити більш широкий спектр застосувань, потенційно розширюючи варіанти використання біткоїна за межі простих транзакцій.
