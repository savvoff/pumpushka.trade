## Огляд

Mastercard запускає нову ініціативу, щоб спростити крипто-транзакції, впроваджуючи перевірені, зрозумілі для людини крипто-псевдоніми для гаманців самостійного зберігання. Ця система використовує Polygon для підтримки в ланцюгу та залучає Mercuryo для перевірки особи.

## Що сталося

Нове впровадження Mastercard має на меті покращити досвід користувачів у сфері криптовалют, замінюючи складні криптографічні адреси простими іменами користувачів. Цей крок розроблено для того, щоб зробити транзакції більш доступними та зручними для осіб, які керують своїми криптоактивами. Використовуючи Polygon, рішення для масштабування другого рівня, Mastercard забезпечує підтримку цих псевдонімів на блокчейні, що полегшує транзакції. Крім того, роль Mercuryo у перевірці особи є важливою для забезпечення безпеки та довіри в цій новій системі.

## Вплив на крипто-ринок

- Спрощує процес відправлення та отримання криптовалют для користувачів.
- Підвищує заходи безпеки через перевірку особи.
- Сприяє більш широкому впровадженню гаманців самостійного зберігання серед користувачів.
- Потенційно підвищує ефективність транзакцій у мережі Polygon.
- Може вплинути на інші фінансові установи для впровадження подібних зручних рішень.
